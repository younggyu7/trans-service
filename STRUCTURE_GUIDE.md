# 통합 번역 플랫폼 - 구조 및 디자인 가이드

## 📋 전체 구조

플랫폼은 4가지 주요 섹션으로 구성됨:
1. **시험 응시** (`/mypage/*`) - 시험 응시자 마이페이지
2. **번역 의뢰** (`/client/*`) - 번역 의뢰인 (클라이언트)
3. **번역가** (`/translator/*`) - 번역 수행자 (번역가)
4. **문제은행** (`/question-bank`) - 문제 관리 및 출제

---

## 1️⃣ 번역 의뢰 (CLIENT) 섹션

### 📂 디렉토리 구조
```
/app/client/
├── requests/           # 내 의뢰 목록 (메인 페이지)
├── request/
│   └── new/           # 새 의뢰 등록 페이지
└── payments/          # 결제 내역 (예상)
```

### 🎨 페이지 레이아웃 구조

#### `/client/requests` (내 번역 의뢰 목록)

**Header 구성:**
- 경로 표시: "홈 > 내 의뢰 목록"
- 제목: "내 번역 의뢰" (큼, 굵은 텍스트)
- 우측에 파란색 "**+ 새 의뢰 등록**" 버튼

**통계 섹션 (4개 카드 그리드):**
```
┌─────────────────┬─────────────────┬─────────────────┬─────────────────┐
│  📋             │  ⏱️             │  🔄             │  ✅             │
│  전체 의뢰      │  대기중         │  진행중         │  완료           │
│  12             │  3              │  2              │  7              │
└─────────────────┴─────────────────┴─────────────────┴─────────────────┘
```
- 각 카드: 흰 배경, 상단 아이콘 이모지, 회색 작은 라벨, 검은 큰 숫자
- 카드 간격: gap-6, 패딩: 6

**탭 필터:**
- 종류: "전체", "대기중", "진행중", "완료"
- 활성 탭: 파란색 아래 테두리 (border-blue-600) + 파란색 텍스트
- 비활성 탭: 회색 텍스트, 호버시 진하게

**의뢰 목록 (카드형):**

각 의뢰 카드 구조:
```
┌─────────────────────────────────────────────────┐
│ 제목                                     [상태배지] │
│ 등록일: 2024-11-20 | 마감일: 2024-12-05         │
│                                                  │
│ 분야: 법률/계약  │  분량: 2,500단어            │
│ 비용: ₩450,000   │  번역사: 김영희 (A등급)    │
│                                                  │
│ [진행 중인 경우] 번역 진행률: 65%                │
│ ████████░░ (프로그레스바)                        │
│                                                  │
│ [상세보기]  [메시지]                           │
└─────────────────────────────────────────────────┘
```

**상태 배지 색상:**
- 진행중: 파란 배경 (bg-blue-100), 파란 텍스트 (text-blue-700)
- 대기중: 노란 배경 (bg-yellow-100), 노란 텍스트 (text-yellow-700)
- 완료: 초록 배경 (bg-green-100), 초록 텍스트 (text-green-700)

**색상 시스템:**
- 주요 색상: 파란색 (#2563EB)
- 통계 카드 배경: 흰색 (bg-white)
- 테두리: 회색 (border-gray-200)
- 배경: 흰색

## 2️⃣ 번역가 (TRANSLATOR) 섹션

### 📂 디렉토리 구조
```
/app/translator/
├── available/        # 신규 번역 요청 (메인 페이지)
├── working/          # 진행 중인 작업
├── completed/        # 완료된 작업
├── earnings/         # 수익 관리
└── profile/          # 프로필 설정
```

### 🎨 페이지 레이아웃 구조

#### `/translator/available` (신규 번역 요청)

**Header 구성:**
- 제목: "신규 번역 요청" (큼, 굵은 텍스트)
- 부제목: "내 전문 분야에 맞는 번역 요청을 찾아보세요" (회색)

**필터 섹션 (4개 컬럼):**
```
┌────────────────┬────────────────┬────────────────┬────────────────┐
│ 언어 조합      │ 전문 분야      │ 분량           │ 긴급           │
│ [Dropdown v]   │ [Dropdown v]   │ [Dropdown v]   │ [Dropdown v]   │
└────────────────┴────────────────┴────────────────┴────────────────┘
```
- 흰 배경, 회색 테두리
- 각 드롭다운: 회색 테두리, 패딩-4

**요청 목록 (카드형):**

각 요청 카드 구조:
```
┌─────────────────────────────────────────────────┐
│ 제목                          ₩480,000           │
│ 기술 문서 번역   [긴급]       견적 금액          │
│ 한국어 → 영어                                    │
│                                                  │
│ 전문 분야: 기술/IT │ 분량: 3,200단어 │ 마감: 2024-12-10 │
│                                                  │
│ [지원하기]  [상세보기]                         │
└─────────────────────────────────────────────────┘
```

**긴급 배지:**
- 빨간 배경 (bg-red-100), 빨간 텍스트 (text-red-700)
- 작은 텍스트 크기

**색상 시스템:**
- 주요 색상: 보라색 (#9333EA) - 번역가 테마
- 가격 텍스트: 보라색 (text-purple-600)
- 버튼: 보라색 배경 (bg-purple-600)
- 배경: 흰색
- 테두리: 회색 (border-gray-200)

**버튼:**
- "지원하기": 보라색 배경, 흰 텍스트, 전체 너비 (flex-1)
- "상세보기": 흰 배경, 회색 테두리, 회색 텍스트

## 🎭 공통 레이아웃 구조

### 모든 섹션에서 사용되는 레이아웃:

```
┌────────────────────────────────────────────────────┐
│  Header (고정)                                     │
├────────────────┬──────────────────────────────────┤
│ IconSidebar    │  메인 콘텐츠 영역                │
│ (아이콘만)     │  - 각 섹션별 레이아웃            │
│ (좁음)         │  - max-width: 1120px 또는 7xl   │
│                │  - 패딩: 8                      │
├─ Sidebar       │                                  │
│ (220px)        │                                  │
│ - 프로필       │                                  │
│ - 메뉴         │                                  │
│ - 지원/통계    │                                  │
└────────────────┴──────────────────────────────────┘
```

### Sidebar 공통 요소:

**프로필 섹션:**
- 원형 프로필 사진 (w-24 h-24)
- 배지 (Premium / 번역가)
- 이름 (큼, 굵음)
- 부연설명 (회색, 작음)

**메뉴:**
- 카테고리 제목 (굵음, 작음)
- 메뉴 항목 (리스트)
  - 활성: 연한 배경색 + 진한 색상 텍스트 + 굵음
  - 비활성: 회색 텍스트 + 호버시 회색 배경

**하단 지원/통계:**
- 번역의뢰: "전문가 신청" 버튼 (보라색)
- 번역가: "이번 달 수익" 카드 (그라디언트 배경, 보라색 텍스트)

## 📐 반복되는 디자인 패턴

### 1. 카드 (Request/Job Cards)
```
bg-white
border border-gray-200
rounded-lg
p-6
hover:shadow-md transition-shadow
```

### 2. 버튼
```
주요 버튼: bg-[색상]-600 text-white rounded-md font-semibold
보조 버튼: border border-gray-300 text-gray-700 rounded-md
```

### 3. 배지
```
px-3 py-1 rounded-full text-xs font-medium
```

### 4. 그리드
```
grid grid-cols-[수] gap-[간격]
```

### 5. 상태 표시
```
배지 + 진행바(진행중인 경우)
```

## ✅ 다음 페이지 구현 시 주의사항

1. **색상 일관성**: 번역의뢰는 파란색, 번역가는 보라색 사용
2. **카드 레이아웃**: 요청/작업 목록은 모두 카드 형식 사용
3. **필터/탭**: 두 섹션 모두 필터링 기능 포함
4. **통계 섹션**: 번역의뢰는 4개 카드 그리드 (의뢰 상태별), 번역가는 필터와 수익 카드 포함
5. **Sidebar 위치**: IconSidebar + 섹션별 Sidebar (220px 고정 너비)
6. **메인 콘텐츠**: max-width 설정 + 중앙 정렬 + 적절한 패딩
